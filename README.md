# KIRD 데이터 분석 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 KIRD 「과학기술 인재개발 활동조사」 재직자 데이터를 활용하여  
과학기술 분야 재직자의 심리적 안전감이 지식공유와 혁신행동에 미치는 영향을 분석한 프로젝트입니다.

분석은 Python을 활용하여 수행하였으며, 기술통계분석, 상관분석, 다중회귀분석을 중심으로 진행했습니다.

## 2. 분석 주제

심리적 안전감이 높은 재직자는 조직 내에서 지식을 더 활발히 공유하고,  
새로운 아이디어를 제안하거나 적용하는 혁신행동도 더 많이 보이는가를 확인했습니다.

## 3. 연구 질문

1. 심리적 안전감은 지식공유에 유의한 영향을 미치는가?
2. 심리적 안전감은 혁신행동에 유의한 영향을 미치는가?

## 4. 사용 데이터

- 데이터명: KIRD 과학기술 인재개발 활동조사 재직자 데이터
- 분석 대상: 과학기술 분야 재직자
- 표본 수: 2,000명
- 변수 수: 312개

## 5. 분석 변수

| 구분 | 변수명 | 설명 |
|---|---|---|
| 독립변수 | 심리적 안전감 | 조직 내에서 실수, 의견 제시, 도움 요청이 안전하게 받아들여지는 정도 |
| 종속변수 1 | 지식공유 | 동료와 정보, 업무 지식, 배움의 기회를 공유하는 정도 |
| 종속변수 2 | 혁신행동 | 새로운 아이디어를 제안하고 업무에 적용하려는 행동 |

## 6. 분석 방법

1. 데이터 불러오기 및 전처리
2. 분석용 subset 생성
3. 척도 신뢰도 분석
4. 기술통계분석
5. 상관분석
6. 다중회귀분석
7. 분석 결과 해석

## 7. 주요 분석 결과

### 7.1 기술통계

| 변수 | 평균 | 표준편차 |
|---|---:|---:|
| 심리적 안전감 | 3.457 | 0.817 |
| 지식공유 | 3.822 | 0.621 |
| 혁신행동 | 3.792 | 0.622 |

### 7.2 상관분석 결과

| 관계 | 상관계수 |
|---|---:|
| 심리적 안전감 - 지식공유 | 0.323 |
| 심리적 안전감 - 혁신행동 | 0.142 |
| 지식공유 - 혁신행동 | 0.314 |

심리적 안전감은 지식공유와 혁신행동 모두와 정적 상관관계를 보였습니다.

### 7.3 회귀분석 결과

| 종속변수 | 심리적 안전감 회귀계수 | 유의확률 | 해석 |
|---|---:|---:|---|
| 지식공유 | 0.244 | p < .001 | 심리적 안전감이 높을수록 지식공유 증가 |
| 혁신행동 | 0.119 | p < .001 | 심리적 안전감이 높을수록 혁신행동 증가 |

## 8. 결론

분석 결과, 심리적 안전감은 지식공유와 혁신행동 모두에 유의한 정적 영향을 미치는 것으로 나타났습니다.

특히 지식공유에 대한 영향이 혁신행동보다 더 크게 나타났습니다.  
이는 구성원이 심리적으로 안전하다고 느낄수록 동료와 지식과 정보를 더 활발히 공유할 가능성이 높다는 점을 보여줍니다.

또한 심리적 안전감은 새로운 아이디어를 제안하고 적용하는 혁신행동과도 관련이 있었습니다.  
따라서 과학기술 분야 조직에서 인재개발과 혁신역량을 높이기 위해서는 교육훈련뿐 아니라 심리적으로 안전한 조직문화 조성이 중요합니다.
